## Default Process
1. Members push code to feature branches and create pull requests
2. All new ad creatives must be copied to the `public/` directory of the repository to be accessible via direct links after deployment.
3. The team lead reviews and merges PRs
4. Infrastructure expert is responsible for pushing these assets to trigger Vercel deployments.
5. Before starting new work, members should pull the latest default branch so they branch from up-to-date code
